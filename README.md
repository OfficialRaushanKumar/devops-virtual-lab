# DevOps CI/CD Virtual Lab

## Project Overview

This project demonstrates an end-to-end DevOps pipeline using:

* Git for version control
* Docker for containerization
* Kubernetes for container orchestration
* Minikube for local Kubernetes cluster

The application is a simple web page deployed using Docker and Kubernetes.


## Steps Performed

1. Created a simple web application
2. Built a Docker image
3. Ran the container locally
4. Created Kubernetes deployment
5. Exposed service using NodePort
6. Deployed application on Minikube
7. Managed project using Git and GitHub
